Из библиотек используютяся только самые необхоидмые это:
1. fmt - для ввывода сообщение
2. os - для чтения с json
3. encoding/json - для маршелинга в json
4. strconv - для конвертации строки в инт
5. strings - для удаление пробелов в тасках (для корректной работы)
6. bufio - для создания сканера и корректной работы с вводимыми данными
